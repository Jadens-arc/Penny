How to Use

Basics

  1. Click New List to make a new list
  2. Click the plus sign to add a new note to the list

Keyboard Shortcuts

Key Stroke Function
Control + w Close Application
Control + n Add Note
Control + Shift + n Add Notes List
Control + r Lock session
Control + Shift + a Toggle Always on top
Control + h Show help screen
Control + d Insert current date
Control + o Open new Wallet
